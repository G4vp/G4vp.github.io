# Pac-Man Deluxe
[Github](https://github.com/G4vp/Pac-Man-Deluxe)
## What I did: 
- Created a modified version of Pac-Man featuring custom power-ups, multi-character support, and procedurally-inspired map designs.
- Leveraged OpenFrameworks to handle real-time rendering, input processing, and asset management in an object-oriented C++ environment.
## Tools:
- C++, OpenFrameworks
