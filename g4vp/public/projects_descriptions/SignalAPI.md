# WebAuthn Signal API for GPM Passkeys in Chrome Desktop
Internship at Google - May 2025 - Aug 2025 
Mountain View, California, United States
## What I did: 
- Implemented a new WebAuthn API using C++ that enables relying parties to report information about existing passkeys back to Google Password Manager, facilitating the update or removal of incorrect or revoked passkeys from provider and system UI. 
- Implemented new pop-ups UI in Chrome to notify users when a passkey was removed or updated via the Signal API using C++, and collaborated with UX and UXW teams to design the UI elements.
## Tools:
- Chromium, C++, JavaScript
