# College Syllabus ChatBot

## **What I did:**
- Designed a relational database schema and engineered a custom ETL pipeline to extract, transform, and load unstructured data from University of Puerto Rico (UPRM) PDF syllabi into a PostgreSQL database.
- Developed a Retrieval-Augmented Generation (RAG) system utilizing Llama 3.2 and a Vector Database to embed academic documents, enabling context-aware responses to student inquiries.
- Built and deployed a REST API using Flask on Heroku to manage data retrieval for both the interactive chatbot and a comprehensive course statistics dashboard.
- Optimized the development environment by containerizing the application and database using Docker, ensuring consistent performance and simplified testing across local and cloud environments.
## Tools:
Python, PostgreSQL, Flask, Llama 3.2, Vector DB, Streamlit, Docker, Heroku
