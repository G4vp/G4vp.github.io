# Black Jack on NES
[Github](https://github.com/G4vp/Super-Black-Jack-NES)
## What I did: 
- Engineered a fully functional Blackjack game for the Nintendo Entertainment System using 6502 Assembly, managing low-level hardware constraints and memory mapping.
- Architected game logic for "the House" AI, implementing automated hit/stay decision-making and real-time bankroll/betting calculations.
- Managed PPU (Picture Processing Unit) registers to handle sprite rendering, background tile mapping, and UI updates within the NES’s limited graphical architecture.
## Tools:
- 6502 Assembly
