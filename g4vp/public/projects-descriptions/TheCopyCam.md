# The Copy Cam
[Github](https://github.com/G4vp/TheCopyCam)
## What I did: 
- Developed a first-person 3D puzzle game featuring a unique duplication mechanic where players clone and manipulate environmental objects to bypass obstacles.
- Created stylized 3D environments and interactive props using Blender, maintaining a consistent visual language and optimized performance.
## Tools:
- Godot 3D, GDScript, Blender
